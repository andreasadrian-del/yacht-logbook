🗄️
Supabase — The Database
Stores everything, handles login, sends live updates
⛵Your Sailing Data Legs, trips, track points, log entries, notes
👤Login Service Secure email/password, data stays private
🔔Live Updates Notifies the app the instant anything changes — no manual refresh needed
📡 Your phone also talks directly to the database — bypassing the server for two things:
→
GPS track points (every 30 seconds while recording) Points go straight to the database for speed. If you lose signal at sea, they're saved on the phone first and uploaded the moment signal returns — nothing is ever lost.
←
Live update subscription (always-on connection) The app keeps a permanent open line to the database. When a leg finishes saving, the Trips screen updates automatically — no need to tap refresh.
